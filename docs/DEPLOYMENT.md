# Deployment Guide

## Networks Supported
- Ethereum Mainnet
- Base
- Optimism
- Arbitrum

## Deployment Steps

1. Configure environment
2. Run deployment script
3. Verify contracts
4. Update frontend/SDK

## Verification

```bash
npx hardhat verify --network <network> <address>
```

## Post-Deployment
- Monitor transactions
- Update documentation
- Announce to community

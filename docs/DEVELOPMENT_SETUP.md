# Development Setup Guide

## Prerequisites
- Node.js >= 18.0.0
- npm or yarn
- Git

## Quick Start

```bash
# Clone the repository
git clone <repo-url>

# Install dependencies
npm install

# Run tests
npm test

# Start development
npm run dev
```

## Environment Variables
Create a `.env` file based on `.env.example`.

## Troubleshooting
Check the issues page for common problems and solutions.

# Frequently Asked Questions

## General

**Q: What networks are supported?**
A: Ethereum, Base, Optimism, Arbitrum, and testnets.

**Q: How do I get testnet tokens?**
A: Use the official faucets for each network.

**Q: Is the code audited?**
A: Check the security section for audit reports.

## Development

**Q: What Node.js version is required?**
A: Node.js 18.0.0 or higher is recommended.

**Q: How do I run tests locally?**
A: Use `npm test` or `yarn test`.

**Q: How do I contribute?**
A: See CONTRIBUTING.md for guidelines.

## Troubleshooting

**Q: Why is my transaction failing?**
A: Check gas settings and contract state.

**Q: Where can I get help?**
A: Open a GitHub issue or join our Discord.

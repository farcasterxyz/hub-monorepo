# Glossary

## Common Terms

**ABI** - Application Binary Interface for smart contracts

**Block** - A batch of transactions bundled together

**DeFi** - Decentralized Finance applications

**EVM** - Ethereum Virtual Machine

**Gas** - Unit measuring computational effort

**Mainnet** - Production blockchain network

**NFT** - Non-Fungible Token

**RPC** - Remote Procedure Call endpoint

**Smart Contract** - Self-executing code on blockchain

**Testnet** - Test network for development

**Transaction** - Signed data package on blockchain

**Wallet** - Software for managing crypto assets

**Wei** - Smallest unit of Ether (10^-18 ETH)
